# CLIw

Your personal command-line toolkit, always at hand.

## What it does

CLIw lets you collect the shell commands you use every day, run them with a click, and see their output — without opening a terminal. Save your most-used flag combinations as presets so you never have to remember the exact syntax again.

## Features (planned)

- **Command library** — organize your commands by category
- **One-click execution** — run any command and watch the output live
- **Parameter presets** — save named flag combinations per command
- **Cross-device** — works on Mac, iPhone, iPad, and Apple Watch

## Getting started

Open the project and run it. A sample command library is included so you can explore the interface right away.

## Status

Early development. The layout and data model are in place. Command execution and persistence are next.
